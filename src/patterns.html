Patterns

Reusable layouts and workflows that combine components into complete user experiences. Each pattern is a structural blueprint for a common user action.

Authentication Flow

The layout structure for login, signup, and forgot-password steps, with form fields and a primary CTA.

Search & Filter

A combination of an input field, dropdown filters, and tag chips for narrowing results.

Onboarding Carousel

A step-by-step wizard pattern with illustration, headline, dot indicators, and "Next" / "Skip" actions.

Data Table with Pagination

A complex pattern combining data rows, selection checkboxes, and page navigation controls.

Empty States

The standardized layout with illustration, headline, supporting text, and a primary action button used when no data exists.

Settings Layout

A standard split-pane pattern with sidebar navigation on the left and form inputs on the right.

Checkout / Cart Summary

A persistent summary block pattern with line items, totals, and a primary CTA used across purchasing steps.
